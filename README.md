## 🛠️ Stack Technique

Ce portfolio a été développé avec une approche moderne axée sur la performance et le design :

* **Framework :** [Angular](https://angular.io/) (v17+)
* **Style :** [Tailwind CSS](https://tailwindcss.com/) pour une interface responsive et épurée.
* **Animations :** Custom CSS 3D Transforms & Transitions pour une expérience utilisateur interactive.
* **Icônes :** [Lucide Icons](https://lucide.dev/) (ou custom SVG).
* **Déploiement :** [Netlify](https://www.netlify.com/) avec intégration continue.
